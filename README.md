## :bulb: Getting Started

#### Clone repository
```
$ git clone https://github.com/BoboLin945/server-log.git
```
#### Install by npm
```
$ npm install
```
#### Execute
```
$ npm run dev
```
#### Browse the website on localhost or you can test on Postman
```
http://localhost:3000/
```
